Simple Space Invaders

Игра по мотиву оригинальной SpaceInvaders

В начале игрок может выбрать уровень. Уровни хранятся отдельно в файле levels.txt
В уровне может быть отличное число противников и преград.
Противник бывает 3-х видов. С 1 одной жизнью - белый, 2 жизни - синий, 3 - красный.
Препятствия - кубик с одной жизнью, желтого цвета.

Игрок белого цвета. У игрока одна жизнь, но 5 попыток. 
Стрелять можно 2 раза в секунду. Противники стреляют случайно, не чаще 2 раз в секунду.
Противиники не могут уничтожить друг друга. При попадании в противника с несколькими жизнями, меняется его цвет.

Размер поля 600 * 800. Размер корабля - 15 * 20 (3 х 4 клетки), в виде квадрата и треугольника. Размер преграды 10*10 (2x2 клетки).

Формат файла уровней:

[номер_уровня]:[количество_элементов]:[Элемент(X,Y,sX,sY),...]

Элементы - E1, E2, E3 - вражеские корабли с 1, 2 и 3 жизнями, W - препятствие

Игровое Поле

Размер клетки - 5*5. Элемент может занимать несколько клеток. Движение возможно по одной клетке.
Игрок движется по горизонтали со скоростью S клеток в секунду, вражеские корабли со скоростью SE клеток в секунду. Снаряд летит со скорость SR клеток в секунду.
